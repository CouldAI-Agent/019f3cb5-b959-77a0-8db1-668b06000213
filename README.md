# Trade Tracker

A Flutter application to manage trading entries and exits with a clear rationale for each trade.

## Features
- Track active trades.
- Log entry price, target exit, and stop loss.
- Record the purpose or thesis for entering each trade.

## Tech Stack
- Flutter
- Material 3 Design

## Setup
Run the app on any supported platform:
```bash
flutter run
```
